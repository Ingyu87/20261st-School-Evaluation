# PRD: 2026학년도 1학기 학교교육과정 운영 평가 결과 분석 웹사이트

> **문서 상태**: 확정 (v0.3) — 검토 체크리스트 전항목 승인  
> **작성일**: 2026-06-25  
> **대상 학교**: 서울가동초등학교  
> **구현 전 사용자 검토 필수**

---

## 1. 제품 개요

### 1.1 목적
2026학년도 1학기 학교교육과정 운영 평가 설문(교원용·교직원용) 응답 데이터를 시각화·해석하여, 학교 구성원이 **한눈에 결과를 파악**하고 **개선 방향을 도출**할 수 있는 단일 페이지 웹 대시보드를 제공한다.

### 1.2 핵심 가치
| 가치 | 설명 |
|------|------|
| **투명성** | 문항별 응답 분포를 수치·차트로 명확히 표시 |
| **통찰** | 문항별·전체 해석으로 데이터 의미 전달 |
| **실행** | 서술형 의견 원문·요약·워드클라우드로 현장 목소리 반영 |
| **일관성** | Clay 디자인 시스템 기반의 전문적·친근한 UI |

### 1.3 범위 (In Scope)
- 교원용 탭 / 교직원용 탭 2개
- 각 탭: 문항별 응답분포, 문항별 해석, 전체 인사이트
- 서술형 주관식: 원본 데이터, 요약, 워드클라우드, 해석
- `json_output/` 기반 정적 데이터 로드
- 반응형 웹 (모바일·태블릿·데스크톱)

### 1.4 범위 외 (Out of Scope — v1)
- 실시간 설문 수집·CSV 업로드 UI
- 사용자 로그인·권한 관리
- PDF/엑셀보내기
- 교원용·교직원용 데이터 교차 비교 탭
- 다국어 지원

## 2. 사용자 및 사용 시나리오

### 2.1 타겟 사용자 ✅ 확정

> **#1 확정 (2026-06-25)**: **교내 전용**

| 사용자 | 사용 목적 |
|--------|-----------|
| 교육과정 부장·교무기획부 | 전체 결과 보고·개선 계획 수립 |
| 교장·부교장 | 경영평가·학교운영위원회 보고 |
| 교원·교직원 | 자체 평가 결과 열람 |

- 외부 공개·URL 공유 범위: 교내 네트워크 / 로컬 `dist` 열람
- 개인정보: 응답자 식별 정보 없음, 타임스탬프 숨김

### 2.2 주요 시나리오
1. **전체 개요 확인**: 홈(히어로) → 탭 선택 → 전체 인사이트 카드로 긍정률·주요 이슈 파악
2. **문항 심층 분석**: 문항 카드에서 분포 차트 + 해석 문구 확인
3. **현장 의견 확인**: 서술형 섹션에서 원문 목록 → 요약 → 워드클라우드 → 해석 순으로 탐색

## 4. 기능 요구사항

### 4.1 공통 레이아웃

```
┌─────────────────────────────────────────────────┐
│  Top Nav: 로고·학교명·평가 기간                    │
├─────────────────────────────────────────────────┤
│  Hero: 제목 + 응답 수·평가 기간 요약               │
├─────────────────────────────────────────────────┤
│  [ 교원용 ]  [ 교직원용 ]   ← category-tab       │
├─────────────────────────────────────────────────┤
│  § 전체 인사이트 (feature cards)                  │
├─────────────────────────────────────────────────┤
│  § 문항별 분석 (18 or 1 cards, 스크롤)            │
├─────────────────────────────────────────────────┤
│  § 서술형 주관식 분석                             │
│    - 원본 데이터 목록                             │
│    - 요약                                         │
│    - 워드클라우드                                   │
│    - 해석                                         │
├─────────────────────────────────────────────────┤
│  Footer                                           │
└─────────────────────────────────────────────────┘
```

### 4.2 탭: 교원용

#### 4.2.1 전체 인사이트
- 총 응답 수 (34명)
- 전체 긍정률 (매우 그렇다+그렇다 비율, 문항별·전체)
- 최고·최저 점수 문항 하이라이트
- 영역별(Ⅰ·Ⅱ·Ⅲ) 평균 비교 (선택)
- 3~5개 핵심 인사이트 문장 (카드 형태)

#### 4.2.2 문항별 분석 (T01~T18, 18개)
각 문항 카드에 포함:
| 요소 | 설명 |
|------|------|
| 문항 번호·영역 배지 | `Ⅲ-2` 등 영역 태그 |
| 문항 전문 | 원문 전체 표시 (접기/펼치기 옵션 검토) |
| 응답 분포 차트 | 막대 또는 도넛 차트 + 건수·% |
| 응답률 | N/34, 무응답 표시 |
| 문항별 해석 | 2~4문장 해석 텍스트 |

#### 4.2.3 서술형 (T19)
| 섹션 | 내용 |
|------|------|
| 원본 데이터 | 타임스탬프(선택)·응답 전문 목록 |
| 요약 | 주제별 bullet 요약 (3~5개) |
| 워드클라우드 | 한국어 키워드 시각화 |
| 해석 | 종합 해석 1단락 |

**교원용 서술형 주요 주제 (데이터 기반 초안)**:
- 학부모 공개수업·학부모총회 일정 의견 수렴
- 교내 체험학습 예산 편성
- 행사 과다·참여 기회 불균형
- 감사·긍정 피드백

### 4.3 탭: 교직원용

구조는 교원용과 동일하되 규모 축소:
- 척도 문항 1개 (S01)
- 서술형 1개 (S02)
- 전체 인사이트·분포·해석·워드클라우드 동일 패턴

### 4.4 척도 점수화 ✅ 확정

> **#2 확정 (2026-06-25)**: **5점 척도 평균** 사용

| 응답 | 점수 |
|------|------|
| 매우 그렇다 | 5 |
| 그렇다 | 4 |
| 보통이다 | 3 |
| 그렇지 않다 | 2 |
| 전혀 그렇지 않다 | 1 (데이터에 없음, 향후 대비) |

- **표시 항목**: 문항별 평균 점수 + 긍정률(매우+그렇다 비율) 병행 표시
- 전체 인사이트에 전체 평균 점수·긍정률 요약 포함

### 4.5 해석 텍스트 생성 방식 ✅ 확정

> **#3 확정 (2026-06-25)**: **JSON 사전 작성 → 사용자 검토 후 수정**

- `interpretations.json`에 문항별·전체·서술형 해석 저장
- 통계 수치(긍정률, 평균 점수)는 자동 산출, 해석 문구는 수동 작성
- 구현 시 빈 템플릿 + 통계 기반 초안 문장 제공 → 검토·수정 워크플로우

### 4.6 서술형 필터 규칙 ✅ 확정

> **#4 확정 (2026-06-25)**: 제안 규칙 적용

**워드클라우드·요약에서 제외**:
- 빈 문자열, 공백만, `.` 단독
- `없음`, `없습니다`, `없습니다.`

**원본 데이터 목록**: 전체 표시 (필터된 항목은 "무의미 응답" 뱃지)

### 4.7 워드클라우드

- 한국어 형태소 분석: `kiwipiepy` (빌드 전처리)
- 불용어: `없음`, `합니다`, `있습니다` 등 기본 세트 + 커스텀
- 색상: Clay 브랜드 컬러 팔레트 적용
- 빌드 시 사전 생성 (`public/data/wordcloud/*.json`)

## 5. 디자인 요구사항

`DESIGN-clay.md` 기반. 교육 평가 대시보드에 맞게 다음을 적용:

### 5.1 색상·분위
- 배경: `#fffaf0` (canvas)
- 카드: pink / teal / lavender / peach / ochre / cream 순환
- 텍스트: ink `#0a0a0a`, muted `#6a6a6a`
- 차트 색: 브랜드 6색 + success/warning (저점수 강조)

### 5.2 타이포
- Display: Inter 500, negative letter-spacing (Plain Black 대체)
- Body: Inter 400
- 문항 전문: body-md, 긴 텍스트는 line-height 1.55

### 5.3 컴포넌트 매핑
| UI 요소 | Clay 컴포넌트 |
|---------|---------------|
| 탭 (교원/교직원) | `category-tab` / `category-tab-active` |
| 전체 인사이트 카드 | `feature-card-pink/teal/lavender...` |
| 문항 카드 | `feature-card-cream` + `product-mockup-card` |
| 서술형 원문 | `testimonial-card` |
| CTA/요약 밴드 | `cta-band-illustrated` |
| 푸터 | `footer` |

### 5.4 일러스트 ✅ 확정

> **#6 확정 (2026-06-25)**: DESIGN-clay 일러스트 **전부 포함**

| 위치 | Clay 컴포넌트 | 구현 |
|------|---------------|------|
| Hero 우측 | `hero-illustration-card` | 3D 클레이 스타일 히어로 일러스트 (학교·어린이 테마) |
| 전체 인사이트 카드 | feature cards | 문항 영역별 마스코트/아이콘 일러스트 |
| CTA/요약 밴드 | `cta-band-illustrated` | 서술형 섹션 상단 장식 일러스트 |
| Footer | `footer` | 지평선형 산(mountain) 일러스트 (Clay 시그니처) |

- Plain Black 웹폰트 없음 → Inter 500 + negative letter-spacing 대체
- 3D 클레이 에셋: SVG/CSS 그라데이션 기반 커스텀 일러스트 (브랜드 6색 활용)

### 5.5 차트 스타일
- 둥근 모서리 (`rounded-lg`)
- cream 배경 위 hairline border
- 애니메이션: 최소 (DESIGN 가이드 준수)

### 5.6 문항 표시 ✅ 확정

> **#10 확정 (2026-06-25)**: **짧은 제목 기본 + "전문 보기" 토글**

- 카드 헤더: 영역 배지 + `shortTitle` (예: "학교 회의 문화 활성화")
- 본문: 접힌 상태 기본, "전문 보기" 클릭 시 `fullText` + 운영 내용 펼침
